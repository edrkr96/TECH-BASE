## mission_6
- 貯金額予測フォームを作成しました。
- 入力された収入や子どもの有無、マイホームの購入などに基づいて将来の貯金額を予測し、グラフとして表示します。
- 特定の年齢における目標金額を設定することができ、結果のページでは、予測と目標額との差額が表示されます。
- セキュリティの問題でログイン情報は非公開としています。
